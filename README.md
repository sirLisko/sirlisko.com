personal (and experimental) website.

### Build
To install all the dependencies:

```
  $ npm install
```

*(after all the dependencies are installed)*

To build the solution:

```
  $ npm run build
```

To run the `dev` environment:

```
  $ npm start
```

To run the tests:

```
  $ npm test
```
